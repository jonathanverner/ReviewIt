About ReviewIt!

ReviewIt is a 'pastebin'-like site which allows pasting snippets of (currently only) python code. Users may attach inline comments to posted snippets. The site also offers three privacy modes for posted snippets:
  • Public — anyone can view and post comments,
  • Semi-public — anyone can view, only people knowing the password may comment and
  • Private — only people knowing the password may comment.

Features

  • Syntaxt hilighting
  • Inline comments
  • No need for signup/login

  • LaTeX rendering
  • PDF export
